# Скрепыш / Clippit

Маленький саркастичный виджет на macOS, вдохновлённый Clippit из Microsoft Office 97.
Живёт в углу экрана, моргает, покачивается и периодически отпускает едкие комментарии
по поводу твоих несделанных задач.

## Что умеет

- Плавающее окошко поверх всех окон, которое можно перетаскивать мышкой.
- Иконка-скрепка в строке меню — оттуда добавляешь задачи и управляешь поведением.
- Каждые ~8 минут Скрепыш сам показывает облачко с напоминанием в иронично-пассивно-агрессивном
  стиле, подмешивая текст реальной задачи из списка.
- Клик по самому Скрепышу = немедленное напоминание (если хочется специально получить порцию
  сарказма).
- Список задач хранится в `~/Library/Application Support/Paperclip/tasks.json`.

## Требования

- macOS 13 (Ventura) или новее
- Swift 5.9+ (входит в Xcode Command Line Tools, `xcode-select --install`)

## Установка как обычного приложения

Самый удобный путь — собрать `.app`-бандл и закинуть в `/Applications`:

```bash
./scripts/build-app.sh        # → dist/Paperclip.app
open dist/                    # перетащи Paperclip.app в /Applications
```

Скрипт сам поднимает release-сборку, кладёт бинарь в правильную структуру
(`Paperclip.app/Contents/MacOS`), генерирует иконку, создаёт `.icns` и
ad-hoc подписывает бандл.  После этого Paperclip — нормальное приложение
macOS: запускается двойным кликом, видно в Spotlight.

> Поскольку приложение **не нотаризовано** Apple, при первом запуске
> macOS покажет предупреждение «приложение от неустановленного
> разработчика». Кликни правой кнопкой → «Открыть» (один раз), либо
> заранее сними карантин:
> ```bash
> xattr -dr com.apple.quarantine /Applications/Paperclip.app
> ```

### DMG для распространения

```bash
./scripts/build-app.sh && ./scripts/make-dmg.sh
# → dist/Paperclip-<version>.dmg
```

DMG содержит сам `Paperclip.app` и symlink на `/Applications`, чтобы
можно было перетащить иконку drag-and-drop.

## Запуск из исходников (без бандла)

```bash
swift run -c release Paperclip
```

Первая сборка занимает ~10–20 секунд, дальнейшие запуски моментальные.

Чтобы остановить — выбери в меню-баре «Выход» (или `Ctrl+C` в терминале, в котором запущено).

## Меню в строке меню

| Пункт                       | Что делает                                                |
| --------------------------- | --------------------------------------------------------- |
| Добавить задачу…            | Открывает окошко со списком и полем ввода                 |
| Список задач…               | То же самое окно                                          |
| Дёрни Скрепыша              | Принудительный комментарий прямо сейчас                   |
| Пауза напоминаний           | Останавливает периодические нагибания                     |
| Спрятать Скрепыша           | Прячет плавающее окошко (он остаётся в строке меню)       |
| Выход                       | Завершает приложение                                      |

## Кастомизация

- Тексты сарказма лежат в `Sources/Paperclip/Quips.swift` — добавляй свои в массивы
  `aboutTask`, `emptyList`, `onAdd`, `onDone`. Шаблон `{task}` подставляет название задачи.
- Период напоминаний — константа `nudgeIntervalSeconds` в `Sources/Paperclip/AppDelegate.swift`
  (по умолчанию `8 * 60` секунд).

## Чтобы запускался при логине

После того как Paperclip.app лежит в `/Applications`:

«Системные настройки → Общие → Объекты входа и расширения → +» —
выбираешь `Paperclip.app`. С этого момента он будет тихо стартовать
вместе с системой.

## Структура

```
Sources/Paperclip/
  main.swift            # точка входа, поднимает NSApplication
  AppDelegate.swift     # окно, статус-бар, таймеры, состояние
  ClippyView.swift      # SwiftUI-вью самого персонажа (глаза, мимика, анимации)
  ClippyShape.swift     # путь скрепки (одна непрерывная проволока)
  SpeechBubble.swift    # жёлтый «стикер» с хвостиком
  TasksWindowView.swift # окно добавления и управления задачами
  TaskStore.swift       # хранение задач в JSON
  Quips.swift           # банк саркастичных фраз

Resources/
  Info.plist            # шаблон бандла (CFBundleIdentifier, LSUIElement и т.д.)

scripts/
  build-app.sh          # собирает release + упаковывает в .app + ad-hoc sign
  make-dmg.sh           # упаковывает .app в drag-and-drop DMG
  render-icon.swift     # рендерит 1024×1024 PNG для AppIcon.icns
```
